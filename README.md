# 🎯 DigiTools - Digital Tools Buying Website

## 📌 Description
DigiTools is a modern digital tools marketplace where users can explore premium tools, add them to cart, and complete purchases easily. The project focuses on clean UI, smooth user experience, and real-time cart functionality.

---

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript (ES6+)
- React.js
- Tailwind CSS
- DaisyUI
- React Toastify

---

## ✨ Features
- 🛒 Add to cart with live navbar count
- ❌ Remove products from cart
- 💰 Total price calculation
- 🔔 Toast notifications (Add, Remove, Checkout)
- 🔄 Toggle between Products and Cart
- 📱 Fully responsive design

---

## 📦 Project Functionality
- Users can browse digital tools
- Add products to cart (no duplicate allowed)
- Remove items from cart
- Proceed to checkout clears all items
- Navbar shows real-time cart count
